Steps to install the plugin
1:- Dowload the custom-form-plugin.php
2:-place it in "/wp-content/plugins/" directory.
3:- Open PLugins on your wordpress website and you will see a plugin named custom form plugin , activate it.
4:- On any page you want to add form just add the short code  [example_contact_form].
5:- Vola !! Its done.
